안녕하세요!
'신입 개발자 박민아' 입니다!

대학을 졸업한 후 하고 싶었던 여러 가지 일들을 하며 미래를 고민하다 보니 개발자로서 늦은 첫 시작을 하게 되었습니다. 자신이 '누군가에게 도움이 되는 가치 있는 사람이 되면 좋겠다.' 생각하게 되었고 생활과 직업에 도움이 되는 편리한 기능을 만드는 개발자가 되기로 결심하였습니다. 이후 국비 지원학원을 다녀 개발 지식과 실무 역량을 강화하였고 현재는 개인적으로 하루에 몇시간씩 프로그래밍 공부에 시간을 투자하고 있습니다.

"저는 이것을 할 수 있습니다."

대학에서 컴퓨터의 기초 지식을 시작으로 Spring, Java, 데이터베이스(MySQL, MariaDB), Android Studio, vs code, Linux 등을 공부했었으며, 최근엔 국비 지원학원을 다니며 전자정부 프레임워크 기반 Spring과 Java를 학습했습니다.

저는 간단한 웹 개발과 디자인을 할 수 있으며, MYSQL을 이용하여 DB를 구축할 수 있습니다. 또한 소스들을 git hub을 통하여 관리하고, 협업이 가능합니다. 현재는 스스로 작은 프로젝트들을 만들며 개발하고 싶었던 서비스를 만들거나, 이미 만들었던 프로젝트들을 보완하거나 기능들을 추가하고 있습니다.

가장 가까운 목표는 단시간에 코딩을 잘 할 수 있도록 개발 능력과 코드의 이해능력을 기르는 것입니다. 그래서 간단한 오픈소스를 찾아 보며 빠르게 코드를 습득할 수 있도록 연습을 하고 있습니다.

"저의 프로젝트 입니다"

다양한 프로젝트를 제작하지는 못했지만, 열심히 공부하고 만들었습니다. 실제 운영되고 있는 웹 사이트들을 참고하며 DB를 설계하고 필요한 기능들을 검색하고 공부하며 개발을 하였습니다. 프로젝트를 진행하며 한땀 한땀 내 손으로 완성되는 기능을 보면서 뿌듯한 감정과 해냈다는 성취감을 얻었습니다.
